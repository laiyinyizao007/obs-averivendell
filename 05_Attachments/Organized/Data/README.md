# Data

Processed data files including spreadsheets, datasets, and configuration files.

## File Naming Convention
- Format: `YYYY-MM-DD_Type_Description_Version.extension`
- Examples:
  - `2024-12-30_Config_ESP32_WiFi_Settings.json`
  - `2024-12-30_Dataset_Job_Market_Analysis.csv`
  - `2024-12-30_Spreadsheet_Project_Budget_v3.xlsx`
  - `2024-12-30_Export_Contacts_LinkedIn.csv`

## Supported Formats
- **Spreadsheets**: .xlsx, .xls, .csv, .tsv
- **Configuration**: .json, .yaml, .toml, .ini
- **Data**: .csv, .tsv, .parquet, .jsonl
- **Database**: .db, .sqlite, .sql
- **Code Data**: .xml, .json, .yaml

## File Types
- **Config** - Configuration files
- **Dataset** - Raw or processed data
- **Spreadsheet** - Excel/Calc files
- **Export** - Exported data from tools
- **Backup** - Backup data files
- **Template** - Reusable templates

## Best Practices
- Include data source in description
- Version important datasets
- Use CSV for simple tabular data
- Consider compression for large files
