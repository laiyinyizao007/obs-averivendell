# Images

Processed images with descriptive names.

## File Naming Convention
- Format: `YYYY-MM-DD_Project_Description.extension`
- Examples:
  - `2024-12-30_Architecture_Diagram.png`
  - `2024-12-30_Meeting_Whiteboard.jpg`
  - `2024-12-30_ESP32_Circuit_Design.png`

## Supported Formats
- PNG (preferred for diagrams)
- JPG (for photos)
- GIF (for animations)
- SVG (for vector graphics)

## Best Practices
- Keep files under 2MB for Git performance
- Use descriptive but concise names
- Include project context in naming
- Optimize images before adding
