# PDFs

Processed PDF documents with descriptive names.

## File Naming Convention
- Format: `YYYY-MM-DD_Category_Description_Version.pdf`
- Examples:
  - `2024-12-30_API_Documentation_v2.pdf`
  - `2024-12-30_Meeting_Notes_Standup.pdf`
  - `2024-12-30_Research_Paper_AI_Architecture.pdf`
  - `2024-12-30_Invoice_Hardware_Components.pdf`

## Categories
- **Documentation** - Technical docs, manuals, guides
- **Research** - Papers, articles, studies
- **Meeting** - Meeting notes, transcripts, recordings
- **Legal** - Contracts, licenses, agreements
- **Finance** - Invoices, receipts, reports
- **Reference** - Datasheets, specifications

## Best Practices
- Compress large PDFs (use tools like PDF Compressor)
- Include version numbers for documentation
- Use descriptive category prefixes
- Consider external storage for files > 10MB
